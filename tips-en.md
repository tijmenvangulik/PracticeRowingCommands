# Tips and tricks

* Use the demo option or the step-by-step option to see exactly what to do
* Follow these steps:
    * Read the commands carefully on the information page
    * Check where the buttons are before you start rowing
    * Move the mouse to the next command before you start
* Take your time, make sure the boat is not going too fast
* Trouble with the mouse? Keyboard control or touch screen works too.
